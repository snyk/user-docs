* [JavaScript](javascript/)
* [PHP](php.md)
* [Python](python.md)&#x20;
* [Ruby](ruby.md)
* [Rust](rust.md)
* [Scala](scala.md)
* [Swift and Objective-C](swift-and-objective-c.md)

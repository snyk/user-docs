# API targets

API targets scan endpoints defined in your API schema (OpenAPI, Postman Collection, or GraphQL). The scanner tests all endpoints from the schema, ensuring complete coverage of your API surface:

* [Configure Postman Collection targets](configure-postman-collection-targets.md) - Set up targets using Postman Collections
* [Configure RAML targets](configure-raml-targets.md) - Set up targets using RAML specifications
* [Configure signed requests](configure-signed-requests.md) - Configure request signing for APIs that require it
* [Configure message level encryption](configure-message-level-encryption.md) - Set up encryption for API message payloads

# Enforce 2FA for all users

Learn how to enforce two-factor authentication (2FA) for all users of your Snyk API & Web account.

Two-factor authentication (2FA) strengthens authentication with an additional layer of security that requires presenting an extra piece of evidence (the possession factor) to an authentication mechanism of a website or application. To obtain the possession factor, you can use an authenticator like Google Authenticator, 1Password, Authy, or Microsoft Authenticator, which provides a random code that changes frequently.

<figure><img src="../../../.gitbook/assets/managing-account-enforce-2fa-for-all-users-overview.png" alt="Two-factor authentication concept"></figure>

If **you are the owner of the Snyk API & Web account**, you can enforce 2FA for all its users so that they have this extra layer of authentication in their profiles.

## Enforce 2FA

In the Snyk API & Web app, enforce 2FA for all the users as follows:

1. Open the **Settings** dropdown menu on the bottom-left corner of the navigation bar and click on **Authentication**.

<figure><img src="../../../.gitbook/assets/managing-account-enforce-2fa-for-all-users-settings-authentication.png" alt="Settings menu with Authentication option"></figure>

1. In the **TWO-FACTOR AUTHENTICATION (2FA) SETTINGS** section, click on **Enforce 2FA**.

<figure><img src="../../../.gitbook/assets/managing-account-enforce-2fa-for-all-users-enforce-button.png" alt="Enforce 2FA button in settings"></figure>

1. A dialog is displayed to enter your password and 2FA code as a security measure.

<figure><img src="../../../.gitbook/assets/managing-account-enforce-2fa-for-all-users-confirmation-dialog.png" alt="Password and 2FA code confirmation dialog"></figure>

Now that you have enforced 2FA, the following happens to the users of your account:

* **Users logging in without 2FA set up for their profile**<br>At login, they must follow some extra steps to set up 2FA for their profile.<br>The procedure will be like steps four and five described in [How to set up 2FA for your profile](https://help.probely.com/en/articles/8945079-how-to-set-up-2fa-for-your-profile).
* **Users already logged in without 2FA set up for their profile**<br>These users will be automatically logged out.<br>When logging back in, they must follow some extra steps to set up 2FA for their profile.<br>The procedure will be like steps four and five described in [How to set up 2FA for your profile](https://help.probely.com/en/articles/8945079-how-to-set-up-2fa-for-your-profile).
* **Users already logged in with 2FA disabled for their profile**<br>Their 2FA will become enabled, and they will be asked to enter the random code generated by the authenticator app installed on their device.
* **Users with 2FA enabled for their profile**<br>They will not notice because they are already using 2FA.

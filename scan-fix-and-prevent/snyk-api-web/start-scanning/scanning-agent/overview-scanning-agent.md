# Scanning agent

The scanning agent is a lightweight component that you install in your network to enable scanning of internal targets. It creates a secure connection between Snyk API & Web and your internal infrastructure, allowing you to scan applications behind firewalls, VPNs, or in private networks:

* [Install a scanning agent](install-scanning-agent.md) - Set up the agent in your environment
* [Scan internal applications](scan-internal-applications.md) - Configure and scan internal targets

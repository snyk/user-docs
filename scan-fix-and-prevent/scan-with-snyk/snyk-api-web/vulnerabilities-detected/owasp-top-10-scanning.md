# OWASP Top 10 scanning

Snyk API & Web can scan for the 2021 and 2025 Open Web Application Security Project (OWASP) Top 10, searching for a wide range of vulnerabilities that belong to the different OWASP Top 10 categories.

## Generate OWASP Top 10 reports

After the target scan, you can generate a target scan report for OWASP Top 10, which includes a table indicating which areas of the Top 10 were **TESTED** and which ones **PASSED** or not.

Visit [Types of target scan reports you can generate with Snyk API & Web](https://help.probely.com/en/articles/2659844-types-of-target-scan-reports-you-can-generate-with-snyk-api-web) for more information about generating reports.

## Scan profiles and OWASP Top 10

All built-in scan profiles check for the Top 10 vulnerabilities, except for the **Lightning** scan profile.

Visit [Built-in scan profiles and their differences](../start-scanning/overview-scan-settings/built-in-scan-profiles.md) for more information about scan profiles.

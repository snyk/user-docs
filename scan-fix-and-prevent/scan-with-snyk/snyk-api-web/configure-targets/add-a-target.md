# Add a target

Add a target to define the scope and behavior of security scans for your web application, website, or API.

A target defines what Snyk API & Web scans and how the scanner behaves. Before adding a target, review the [target overview](what-is-a-target.md) to understand target concepts.

1. Navigate to **Targets** and click **Add**.
2. Complete the **Add target** form with your target details.
3. Click **Add** to create the target.

During this process, Snyk API & Web checks connectivity to your target. If the target is unreachable, you can bypass the warning and add the target anyway. However, the target must be reachable for scans to run successfully.

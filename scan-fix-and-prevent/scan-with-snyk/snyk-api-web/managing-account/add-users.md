# Add users

Learn how to add team members to your Snyk API & Web account.

## Add a user to your account

To add users to your account:

1. Open the **Settings** dropdown menu on the bottom-left corner of the navigation bar.

<figure><img src="../../../.gitbook/assets/add-users-settings-menu.png" alt="Settings dropdown menu in navigation bar"></figure>

2. Select **Users**.
1. Click **Add User**.
1. Enter the following information for your team member:
   * Name
   * Email
   * Title
1. Configure the intended Role and Scope.
1. Optionally, set up Labels to help you filter users later.
1. Click **Add**.

You can repeat this process to continue adding colleagues to Snyk API & Web.

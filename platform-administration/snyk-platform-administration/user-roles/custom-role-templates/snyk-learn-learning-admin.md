# Snyk Learn - Learning Admin

{% hint style="info" %}
**Feature availability**

This role requires the [Snyk Learn Assignments](https://app.gitbook.com/o/-M4tdxG8qotLgGZnLpFR/s/L7HyJj9FsK1W4pNt8Gzl/discover-snyk/snyk-learn/snyk-learn-assignments) feature, which is part of the Learning Management add-on.  For more information, contact your Snyk account team.
{% endhint %}

To use [Snyk Learn Access Controls](https://app.gitbook.com/o/-M4tdxG8qotLgGZnLpFR/s/L7HyJj9FsK1W4pNt8Gzl/discover-snyk/snyk-learn/snyk-learn-access-controls) beyond the default Snyk Group and Organization Admin roles, a Snyk Group Admin must create a custom role.

To create this role, enable the following permissions in the relevant categories:

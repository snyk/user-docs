# Overview

## Cloud AI platforms

Snyk Evo integrates with cloud AI platforms to find AI assets deployed outside your codebase and developer machines. These assets appear in your Evo inventory alongside code and machine-layer findings and include:

* AI models your workforce actively uses
* Organization-approved MCP servers
* Tool-level permission restrictions your admins set

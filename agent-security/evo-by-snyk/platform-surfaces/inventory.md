# Inventory

Inventory surfaces the AI assets discovered across your environment alongside the assessed risk so you can review it in context.

* [AI-SPM](https://app.gitbook.com/s/N5N885PkllOWeBmgm3Bp/evo-by-snyk/ai-spm/ai-asset-visibility) discovers assets in your source code through a source code management (SCM) integration.
* [Agent Supply Chain Security](https://app.gitbook.com/s/N5N885PkllOWeBmgm3Bp/evo-by-snyk/agentic-development-security-ads/agent-supply-chain-security) within ADS discovers the components agents use on end users' machines.

You can explore Inventory with Evo chat. For example, ask which assets occur most often in your code.

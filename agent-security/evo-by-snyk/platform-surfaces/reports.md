# Reports

Save views generated by Evo chat as reports to track changing data. Reports update automatically as data changes.

#### Create a report

To create a report:

1. Run a query using Evo chat to display issues based on your criteria.
2. Click **Save report**.
3. Enter a name for the report.
4. Click **Save**.

The report appears in **Saved Reports**.

#### Delete a report

To delete a report:

1. Navigate to **Reports**.
2. In the table view, click the garbage bin icon for the report you want to delete. A confirmation dialog box appears.
3. Click **Delete**.

You cannot undo this action, but you can regenerate the report using Evo chat.

#### Example prompts

Click **Generate report with Evo** in the Evo chat to see prompt ideas, such as:

* Show me all models grouped by count
* List all critical issues
* Show me AI assets that have issues

You can also try these prompts:

* Which policies violate the most assets?
* Show the model that violates the most policies
* Show me all assets violating low-severity policies
* What asset type has the most policy violations?

For more ideas, ask the Evo chat: "Show me more ideas for reports".
